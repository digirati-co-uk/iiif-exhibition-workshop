This workshop is a guided introduction to IIIF: what it is, how to find and view IIIF content, and how to create your own manifests (and optionally collections and exhibitions) using freely available tools.

We will guide the session following the planned agenda, but you can use the links below to find your way through this website content. You can always return here after the workshop and step through the activities in your own time.
