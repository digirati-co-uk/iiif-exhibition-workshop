To get started, so that you can participate in the workshop activities, in your browser navigate to the Digirati [IIIF Cloud Service](https://management.dlcs-trial.digirati.io/) workshop portal.

:::note
You can use the 'IIIF Workshop portal' menu link which is available in the top right of this documentation site.
:::

If you have not yet logged into your account, please follow the steps below.

## Access to your account

A IIIF Cloud Services account has been created for workshop participants using the email address you provided for the conference. You will have received instructions on how to access your account and reset your password to login. Please ask the Digirati team for support during the workshop if you have any access issues.

![Select the option to Login](https://digirati-workshop-iiif-2025.pages.dev/login-to-iiif-workshop-portal.PNG)

On login, you should then be navigated to https://management.dlcs-trial.digirati.io/Services/IIIF/Home; if not, click on the IIIF Cloud Services icon on the top left of the screen to be redirected to the home page.

![Workshop homepage](https://digirati-workshop-iiif-2025.pages.dev/successful-login-to-workshop-portal.PNG)
